# Завдання 1. Улюблені мелодії

## Умова завдання

Написати програму **«Улюблені мелодії»**. У список `ComboBox` потрібно додати 10 назв музичних творів. Вибрані елементи потрібно вивести в `ListBox`.

## Опис реалізації

У першому проєкті реалізовано форму, на якій розміщено список `ComboBox` із десятьма музичними творами. Користувач може обрати один твір зі списку та натиснути кнопку **«Додати»**.

Після натискання кнопки вибрана мелодія додається до `ListBox`. Якщо така мелодія вже була додана, програма виводить повідомлення, що цей елемент уже є у списку.

## Використані елементи

| Елемент    | Призначення                                    |
| ---------- | ---------------------------------------------- |
| `ComboBox` | Вибір музичного твору зі списку                |
| `ListBox`  | Виведення вибраних мелодій                     |
| `Button`   | Додавання, очищення списку та вихід з програми |
| `Label`    | Підписи до елементів форми                     |

## Список мелодій

У програмі використано такі музичні твори:

```text
Океан Ельзи — Без бою
The Weeknd — Blinding Lights
Imagine Dragons — Believer
Queen — Bohemian Rhapsody
Nirvana — Smells Like Teen Spirit
Metallica — Nothing Else Matters
Linkin Park — Numb
Ricky Martin — Livin La Vida Loca
Pink Floyd — Another Brick in the Wall
Daft Punk — Get Lucky
```

## Перевірка роботи

1. Запустити проєкт `Task1_FavoriteMelodies`.
2. Обрати мелодію у списку `ComboBox`.
3. Натиснути кнопку **«Додати»**.
4. Перевірити, що вибрана мелодія з’явилася у `ListBox`.
5. Натиснути кнопку **«Очистити»**.
6. Перевірити, що список вибраних мелодій очищено.

# Завдання 2. Пошук роботи

## Умова завдання

Написати програму **«Пошук роботи»**. У списку `ListBox` потрібно вказати тип роботи:

* системний адміністратор;
* веб-програміст;
* веб-дизайнер;
* модератор сайтів;
* інші варіанти.

Список потрібно відсортувати. У списку `ComboBox` потрібно вказати початкову зарплату. За допомогою `NumericUpDown` потрібно вибрати стаж роботи. Результати роботи потрібно вивести в текстове поле.

## Опис реалізації

У другому проєкті створено програму для пошуку роботи. Користувач обирає тип роботи зі списку `ListBox`, початкову зарплату зі списку `ComboBox` і стаж роботи за допомогою `NumericUpDown`.

Список професій у `ListBox` відсортовано за алфавітом. Після натискання кнопки **«Показати результат»** програма виводить обрані дані в багаторядкове текстове поле.

Додатково реалізовано приблизний розрахунок зарплати з урахуванням стажу роботи.

## Використані елементи

| Елемент         | Призначення                         |
| --------------- | ----------------------------------- |
| `ListBox`       | Вибір типу роботи                   |
| `ComboBox`      | Вибір початкової зарплати           |
| `NumericUpDown` | Вибір стажу роботи                  |
| `TextBox`       | Виведення результату                |
| `Button`        | Показ результату, очищення та вихід |

## Список професій

У програмі використано такі типи роботи:

```text
Системний адміністратор
Веб-програміст
Веб-дизайнер
Модератор сайтів
Тестувальник ПЗ
DevOps інженер
UI/UX дизайнер
Контент-менеджер
Бізнес-аналітик
Frontend-розробник
```

## Початкова зарплата

У `ComboBox` додано такі варіанти початкової зарплати:

```text
10000 грн
15000 грн
20000 грн
25000 грн
30000 грн
35000 грн
40000 грн
50000 грн
```

## Формула розрахунку

У програмі використано умовну формулу:

```text
Надбавка за стаж = стаж роботи × 1000 грн
Орієнтовна зарплата = початкова зарплата + надбавка за стаж
```

## Приклад роботи

Вхідні дані:

```text
Тип роботи: Веб-програміст
Початкова зарплата: 20000 грн
Стаж роботи: 3 роки
```

Розрахунок:

```text
Надбавка за стаж = 3 × 1000 = 3000 грн
Орієнтовна зарплата = 20000 + 3000 = 23000 грн
```

Очікуваний результат:

```text
Результати пошуку роботи:
Тип роботи: Веб-програміст
Початкова зарплата: 20000 грн
Стаж роботи: 3 років
Надбавка за стаж: 3000 грн
Орієнтовна зарплата: 23000 грн
```

## Перевірка роботи

1. Запустити проєкт `Task2_JobSearch`.
2. Обрати тип роботи у `ListBox`.
3. Обрати початкову зарплату у `ComboBox`.
4. Вказати стаж роботи через `NumericUpDown`.
5. Натиснути кнопку **«Показати результат»**.
6. Перевірити, що дані виведено у текстове поле.
7. Натиснути **«Очистити»** і перевірити, що форма повернулася до початкового стану.

# Завдання 3. Улюблені мелодії через CheckedListBox

## Умова завдання

Модифікувати завдання 1. Список музичних творів потрібно розмістити в `CheckedListBox`. Вибрані елементи потрібно вивести в `ListBox`.

## Опис реалізації

У третьому проєкті модифіковано програму **«Улюблені мелодії»**. Замість `ComboBox` використано `CheckedListBox`.

Користувач може поставити галочки біля кількох музичних творів. Після натискання кнопки **«Вивести вибрані»** усі позначені елементи виводяться у `ListBox`.

Також реалізовано кнопку **«Очистити»**, яка знімає всі позначки в `CheckedListBox` і очищає `ListBox`.

## Використані елементи

| Елемент          | Призначення                           |
| ---------------- | ------------------------------------- |
| `CheckedListBox` | Вибір кількох музичних творів         |
| `ListBox`        | Виведення вибраних елементів          |
| `Button`         | Виведення вибраних, очищення та вихід |
| `Label`          | Підписи до елементів форми            |

## Список мелодій

У `CheckedListBox` додано 10 музичних творів:

```text
Океан Ельзи — Без бою
The Weeknd — Blinding Lights
Imagine Dragons — Believer
Queen — Bohemian Rhapsody
Nirvana — Smells Like Teen Spirit
Metallica — Nothing Else Matters
Linkin Park — Numb
Ricky Martin — Livin La Vida Loca
Pink Floyd — Another Brick in the Wall
Daft Punk — Get Lucky
```

## Перевірка роботи

1. Запустити проєкт `Task3_CheckedMelodies`.
2. Поставити галочки біля кількох музичних творів.
3. Натиснути кнопку **«Вивести вибрані»**.
4. Перевірити, що вибрані мелодії з’явилися у `ListBox`.
5. Натиснути кнопку **«Очистити»**.
6. Перевірити, що всі галочки знято, а список `ListBox` очищено.

# Опис кнопок

| Кнопка               | Дія                                                        |
| -------------------- | ---------------------------------------------------------- |
| `Додати`             | Додає вибрану мелодію до списку                            |
| `Вивести вибрані`    | Виводить позначені елементи з `CheckedListBox` у `ListBox` |
| `Показати результат` | Виводить результат пошуку роботи                           |
| `Очистити`           | Очищає введені або вибрані дані                            |
| `Вихід`              | Закриває форму                                             |

# Висновок

Під час виконання практичного заняття було створено рішення `PracticeWork4`, яке містить три окремі Windows Forms проєкти.

У першому проєкті реалізовано програму **«Улюблені мелодії»**, у якій використано `ComboBox` для вибору музичного твору та `ListBox` для виведення вибраних елементів.

У другому проєкті створено програму **«Пошук роботи»**. У ній використано `ListBox` для вибору типу роботи, `ComboBox` для вибору початкової зарплати, `NumericUpDown` для вибору стажу роботи та `TextBox` для виведення результатів.

У третьому проєкті модифіковано першу програму. Список музичних творів розміщено в `CheckedListBox`, що дозволяє обирати одразу кілька елементів. Вибрані елементи виводяться у `ListBox`.

У результаті було закріплено навички роботи з елементами керування списками у Windows Forms: `ListBox`, `ComboBox` та `CheckedListBox`, а також навички створення обробників подій для кнопок.
